ENERCON SOLUTIONS WEBSITE
=========================

File Structure:
enercon-website/
├── index.html              # Home page
├── about.html              # About Us page
├── electrical.html         # Electrical Division page
├── civil.html             # Civil Division page
├── contact.html           # Contact Us page
├── privacy-policy.html    # Privacy Policy page
├── terms.html             # Terms & Conditions page
├── css/
│   ├── style.css         # Main stylesheet
│   └── responsive.css    # Responsive styles
├── js/
│   ├── main.js           # Main JavaScript
│   └── analytics.js      # Analytics tracking (optional)
└── assets/
    └── (your logo files)

HOW TO USE:
1. Save all files in the structure shown above
2. Replace logo-placeholder with your actual logo:
   - Save your logo as 'logo.png' in the assets folder
   - Update the logo code in each HTML file
3. Replace Google Analytics ID:
   - Sign up at analytics.google.com
   - Get your tracking ID (starts with G-)
   - Replace 'G-XXXXXXXXXX' in index.html with your ID
4. To run the website:
   - Open index.html in any web browser
   - All pages are linked and will work
